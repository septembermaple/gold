# 黄金市场数据分析与信号分析系统

## 概述

本 Skill 提供完整的黄金市场分析框架,包括:
- **十维度环境背景分析**: 地缘风险、股市波动、期货持仓、技术面、实际利率、通胀、就业、汇率、美债赤字、央行购金
- **实时数据获取**: 黄金价格、ETF数据、宏观经济指标
- **投资信号评分**: 基于多维度分析生成买入/持有/卖出信号
- **个人化建议**: 结合用户背景提供定制化投资建议

## 安装依赖

```bash
pip install yfinance requests pandas
```

## 可选配置

### FRED API (获取美国宏观经济数据)

1. 访问 https://fred.stlouisfed.org/docs/api/api_key.html
2. 申请免费 API key
3. 在 `scripts/get_macro_data.py` 中替换 `your_api_key_here`

## 使用脚本

### 获取黄金价格数据
```bash
python scripts/get_gold_data.py
```

输出:
- 伦敦现货黄金价格
- 黄金9999价格
- SPDR黄金ETF持仓
- 华安黄金ETF数据

### 获取宏观经济数据
```bash
python scripts/get_macro_data.py
```

输出:
- 美债收益率(2年/10年/30年)
- CPI/PCE通胀数据
- 非农就业、失业率
- 美元指数
- 人民币汇率
- VIX恐慌指数

### 生成完整分析报告
```bash
python scripts/get_all_data.py
```

输出:
- 所有市场数据
- 十维度评分分析
- 投资信号(买入/持有/卖出)
- JSON格式报告

## 评分模型

### 十维度权重分配

**短期交易属性** (40%)
- 地缘风险: 10%
- 股市波动: 10%
- 期货持仓: 10%
- 技术面: 10%

**中期金融属性** (40%)
- 实际利率: 15%
- 通胀: 15%
- 就业: 10%

**长期货币属性** (20%)
- 人民币汇率: 8%
- 美债赤字: 6%
- 央行购金: 6%

### 评分规则
- **利好金价**: 1分
- **中性/不变**: 0.5分
- **利空金价**: 0分

### 信号生成
- 总分 ≥ 7分: 强烈买入
- 总分 5-7分: 买入/增持
- 总分 4-5分: 持有观望
- 总分 3-4分: 减持/谨慎
- 总分 ≤ 3分: 卖出/规避
