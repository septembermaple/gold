## 一、项目概述

### 1.1 目标

构建一个**单页 Web Dashboard**，整合黄金核心驱动因子数据，为投资者提供：

- 不同黄金投资标的（实物黄金、黄金ETF、黄金期货、金矿股）的**当前风险等级**
- **市场情绪**综合评分（贪婪/恐惧）
- 各因子**利好/利空**信号一览
- **波动率**监控
- **短期（1-4周）/ 中期（1-6月）/ 长期（1年+）走势预期**

### 1.2 用户画像

| 角色 | 需求 |
|------|------|
| 个人投资者 | 快速判断当前黄金市场状态，辅助买卖决策 |
| 宏观研究者 | 一屏看全关键宏观因子，追踪因子共振 |
| 短线交易者 | 关注波动率、持仓情绪、技术面信号 |

### 1.3 技术选型（建议）

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML + Tailwind CSS + Chart.js / ECharts | 静态单页，轻量化 |
| 数据获取 | Python 脚本（定时任务） | 从 FRED API、CFTC、WGC 等获取数据 |
| 数据存储 | JSON 文件（`data/` 目录） | 前端直接 fetch JSON，无需后端服务 |
| 部署 | GitHub Pages / Vercel | 零成本静态托管 |

## 二、数据源与指标定义

### 2.1 核心数据源

| # | 指标 | FRED/数据源 ID | 频率 | 用途 |
|---|------|---------------|------|------|
| 1 | 10Y TIPS 实际利率 | `DFII10` (FRED) | 日度 | 黄金最核心驱动因子 |
| 2 | 10Y 名义利率 | `DGS10` (FRED) | 日度 | 利率环境判断 |
| 3 | 10Y 通胀预期（Breakeven） | `T10YIE` (FRED) | 日度 | 通胀预期追踪 |
| 4 | 美元指数（贸易加权） | `DTWEXBGS` (FRED) | 日度 | 美元强弱 |
| 5 | 核心CPI（同比） | `CPILFESL` (FRED) | 月度 | 通胀水平 |
| 6 | 核心PCE（同比） | `PCEPILFE` (FRED) | 月度 | 美联储首选通胀指标 |
| 7 | 黄金价格（LBMA AM Fix） | `GOLDAMGBD228NLBM` (FRED) | 日度 | 基准金价 |
| 8 | 美联储总资产 | `WALCL` (FRED) | 周度 | 流动性追踪（QE/QT） |
| 9 | CFTC COT（COMEX 黄金） | CFTC 官方 CSV | 周度 | 投机情绪 / 持仓结构 |
| 10 | GLD ETF 持仓量 | SPDR 官方 / 第三方 | 日度 | 资金流向 |
| 11 | VIX 恐慌指数 | `VIXCLS` (FRED) | 日度 | 市场风险偏好 |
| 12 | 联邦基金利率 | `DFF` (FRED) | 日度 | 当前利率水平 |

### 2.2 衍生计算指标

| 指标 | 计算方式 | 说明 |
|------|----------|------|
| 实际利率变动趋势 | DFII10 的 20日/60日移动均线 & 斜率 | 判断利率环境方向 |
| 美元动量 | DTWEXBGS 20日 ROC（变化率） | 美元走强/走弱动量 |
| 金价波动率 | 黄金价格 20日年化标准差 | 短期风险度量 |
| 金价技术趋势 | 50日/200日均线交叉 | 金叉/死叉信号 |
| COT 投机净多比例 | 净多头 / 总持仓 | 市场情绪过热判断 |
| 实际利率-金价背离度 | 两者 Z-Score 差异 | 寻找超买/超卖信号 |
| ETF 资金流动量 | GLD 持仓 5日变化率 | 短期资金流向 |

### 3.4 模块 [C] — 风险与情绪仪表盘

右侧区域，含两个可视化组件：

#### C1: 情绪仪表盘（Gauge）

- 综合市场情绪得分（0-100）
- 分档配色：极度恐惧 → 恐惧 → 中性 → 贪婪 → 极度贪婪

**情绪综合评分算法（建议权重）：**

| 因子 | 权重 | 利好黄金（偏"恐惧"端） | 利空黄金（偏"贪婪"端） |
|------|------|------------------------|------------------------|
| VIX | 15% | VIX > 25 | VIX < 15 |
| 实际利率趋势 | 25% | 下行 | 上行 |
| 美元指数趋势 | 20% | 下行 | 上行 |
| COT 投机净多比例 | 15% | 低位（有加仓空间） | 极高位（拥挤） |
| ETF 资金流 | 15% | 持续流入 | 持续流出 |
| 金价技术趋势 | 10% | 均线多头排列 | 均线空头排列 |

> 注意：此处的"贪婪/恐惧"是从**黄金投资者视角**——贪婪 = 市场对黄金过度乐观（可能需要警惕回调），恐惧 = 市场对黄金过度悲观（可能是买入机会）。

#### C2: 驱动因子雷达图

六轴雷达图，每轴代表一个核心驱动因子对黄金的支撑强度（0-10）:

- 实际利率（低 = 高分）
- 美元强弱（弱 = 高分）
- 通胀预期（高 = 高分）
- 流动性（宽松 = 高分）
- 避险需求（高 = 高分）
- 资金流入（强 = 高分）

### 3.6 模块 [E] — 不同投资标的风险矩阵

表格形式，按投资标的分行：

| 投资标的 | 当前风险等级 | 核心关注因子 | 风险信号 | 机会信号 | 建议仓位 |
|----------|------------|------------|---------|---------|---------|
| 实物黄金 | 🟢低/🟡中/🔴高 | 长期通胀 + 央行购金 | ... | ... | 保守/适中/激进 |
| 黄金ETF (GLD) | 🟢/🟡/🔴 | 实际利率 + 资金流 | ... | ... | ... |
| 黄金期货 | 🟢/🟡/🔴 | 利率预期 + 持仓结构 + 波动率 | ... | ... | ... |
| 金矿股 (GDX) | 🟢/🟡/🔴 | 金价 + 油价 + 股市 | ... | ... | ... |

**风险等级算法：**

每个标的根据其侧重因子加权计算：

- **实物黄金**：60%长期通胀趋势 + 30%央行购金趋势 + 10%金价技术面
- **黄金ETF**：40%实际利率 + 30%资金流 + 20%美元 + 10%技术面
- **黄金期货**：30%利率预期 + 25%COT持仓 + 25%波动率 + 20%技术面
- **金矿股**：30%金价趋势 + 25%股市环境(VIX) + 25%油价 + 20%美元

### 3.7 模块 [F] — 走势预期

三列卡片，分别展示**短期 / 中期 / 长期**预期：

| 维度 | 时间范围 | 核心依据因子 | 展示内容 |
|------|----------|------------|---------|
| 短期 | 1-4 周 | 技术面 + COT情绪 + 近期事件日历 | 方向(↑↓→) + 置信度 + 关键支撑/阻力位 |
| 中期 | 1-6 月 | 实际利率趋势 + 美元趋势 + 美联储政策路径 | 方向 + 目标区间 + 核心驱动逻辑 |
| 长期 | 1年+ | 通胀趋势 + 央行购金 + 全球流动性 | 方向 + 结构性因素总结 |

每张卡片包含：
- **方向指示**：看多 🟢 / 中性 🟡 / 看空 🔴
- **置信度**：高/中/低
- **关键因子列表**（带当前值）
- **一句话总结**

**走势预期判定逻辑：**

短期预期：
- 技术面金叉 + COT未过热 + 近期无鹰派事件 → 看多
- 技术面死叉 + COT极度拥挤 → 看空
- 信号矛盾 → 中性

中期预期：
- 实际利率下行趋势 + 美元走弱 + 降息预期 → 看多
- 实际利率上行 + 美元走强 + 加息预期 → 看空

长期预期：
- 央行持续购金 + 通胀中枢上移 + 全球流动性扩张 → 看多
- 通缩风险 + 强势美元周期 → 看空

## 四、数据更新策略

| 数据类型 | 更新频率 | 实现方式 |
|----------|----------|---------|
| FRED 日度数据 | 每日 1 次（美东 18:00 后） | Python cron / GitHub Actions |
| CFTC COT | 每周五 15:30 ET 后 | Python 定时抓取 |
| GLD 持仓 | 每日 1 次 | 爬虫或 API |
| 页面更新 | 数据刷新后自动重新生成 | JSON 写入 → 前端 fetch |

### 4.1 数据获取流程

```
[定时任务] 
    → Python 脚本调用 FRED API (api key)
    → 获取最新数据
    → 计算衍生指标（均线、波动率、信号等）
    → 输出 JSON 文件（data/dashboard.json）
    → 前端加载 JSON → 渲染 Dashboard
```

## 五、UI/UX 要求

### 5.1 视觉风格

- **主色调**：深色背景（#1a1a2e）+ 金色点缀（#f0c040）
- **辅助色**：
  - 利好/看多：#22c55e（绿）
  - 利空/看空：#ef4444（红）
  - 中性：#eab308（黄）
  - 信息：#3b82f6（蓝）
- **字体**：系统默认无衬线（-apple-system, 'Segoe UI', sans-serif）
- **风格**：金融终端感（Bloomberg-like），信息密度适中

### 5.2 响应式

- 桌面优先（1280px+），两栏布局
- 平板（768px-1279px），单栏堆叠
- 手机（<768px），卡片竖排，图表可横滑

### 5.3 交互

- 图表支持 hover 显示数据点详情
- 时间范围可切换
- 所有数据卡片可点击展开详细说明
- 页面顶部显示"最后更新时间"
