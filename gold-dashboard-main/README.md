# 黄金投资 Dashboard — 数据说明与解读指南

## 项目概览

本项目是一个黄金投资多因子分析 Dashboard，通过 Python 脚本从多个官方数据源拉取实时宏观数据，经过信号引擎、情绪评分、风险矩阵等模型计算后，输出 JSON 并在前端单页面展示。

```
fetch_data.py  →  data/dashboard.json  →  index.html (Chart.js 可视化)
```

---

## 一、数据来源分类

Dashboard 中的数据分为三层：**真实外部数据** → **标准计算指标** → **自定义分析模型**。

### 1.1 真实外部数据（API 实时拉取）

所有原始数据均来自官方或半官方渠道，无任何人为捏造：

| 数据 | 代码/来源 | 频率 | 说明 |
|------|----------|------|------|
| 金价 | yfinance `GC=F` (COMEX期货) | 日度 | COMEX 黄金期货连续合约收盘价 |
| 10Y TIPS 实际利率 | FRED `DFII10` | 日度 | **黄金最核心的驱动因子**，与金价高度负相关 |
| 10Y 名义利率 | FRED `DGS10` | 日度 | 美国国债收益率基准 |
| 10Y Breakeven 通胀预期 | FRED `T10YIE` | 日度 | 名义利率 − 实际利率，反映市场通胀预期 |
| 美元指数 | FRED `DTWEXBGS` | 日度 | 广义贸易加权美元指数（Fed 发布） |
| 核心 CPI 指数 | FRED `CPILFESL` | 月度 | 剔除食品和能源的消费者价格指数 |
| 核心 PCE 指数 | FRED `PCEPILFE` | 月度 | 美联储首选通胀指标 |
| 美联储总资产 | FRED `WALCL` | 周度 | 反映 QE/QT 流动性状态 |
| VIX 恐慌指数 | FRED `VIXCLS` | 日度 | 标普500期权隐含波动率 |
| 联邦基金利率 | FRED `DFF` | 日度 | 美联储政策利率 |
| CFTC COT 持仓 | CFTC `deafut.txt` + 年度 ZIP | 周度 | COMEX 黄金期货投机/商业净多头 |
| GLD ETF | yfinance `GLD` | 日度 | SPDR Gold Shares AUM、NAV、成交量 |
| 油价 (WTI) | yfinance `CL=F` (WTI期货) | 日度 | WTI 原油期货连续合约，用于矿股成本分析 |
| 白银价格 | yfinance `SI=F` (COMEX白银期货) | 日度 | 用于计算金银比 |
| GDX 金矿ETF | yfinance `GDX` | 日度 | VanEck Gold Miners ETF 价格，用于矿股风险评估 |
| 5Y5Y 远期通胀预期 | FRED `T5YIFR` | 日度 | 5年期5年远期通胀预期，长期通胀趋势指标 |
| 收益率曲线 (10Y-2Y) | FRED `T10Y2Y` | 日度 | 10Y-2Y 国债利差，衰退/加息周期代理指标 |
| 央行购金 | WGC 季度报告 (手动) | 季度 | 全球央行净购金量（吨），结构性需求因子 |

> **FRED API**：需设置环境变量 `FRED_API_KEY`，可在 https://fred.stlouisfed.org/docs/api/api_key.html 免费申请。

### 1.2 标准计算指标（公式透明，业界通用）

基于原始数据用标准金融公式计算，无自定义成分：

| 指标 | 公式 | 用途 |
|------|------|------|
| MA20 / MA50 / MA60 / MA200 | 简单移动平均 `mean(最近N日)` | 趋势判断、金叉死叉 |
| RSI(14) | Wilder 相对强弱指标 | 超买(>70)/超卖(<30) |
| 20日变化率 (ROC) | `(当前价/20日前价 − 1) × 100` | 动量衡量 |
| 年化波动率 | `std(对数收益率) × √252 × 100` | 风险度量 |
| 核心 CPI/PCE 同比 | `(当月值/12月前值 − 1) × 100` | 通胀趋势 |
| 美联储总资产周环比 | `(本周/上周 − 1) × 100` | 扩/缩表判断 |
| 金叉/死叉 | MA50 > MA200 为金叉，反之为死叉 | 中长期趋势信号 |
| Z-Score 背离度 | `Z(gold) − Z(−realYield)`, 60日窗口 | 实际利率与金价背离程度 |
| 金银比 | `金价 / 白银价格` | 经典估值指标，均值回归特性 |
| COT 净多头 | `NonCommercial Long − NonCommercial Short` | 投机资金方向 |
| COT 百分位 | 当前净多头在近3年周度数据中的排名百分比 | 衡量仓位极端程度 |
| GLD 5日资金流 | `近5日 dollarVolume / 前5日 dollarVolume − 1` | 近期资金流入/流出 |
| 支撑/阻力位 | 近60日价格的 P10/P25（支撑）, P75/P90（阻力） | 关键价位参考 |

### 1.3 自定义分析模型（阈值、权重、规则由脚本定义）

以下部分是我们基于黄金投资逻辑**自行设计**的分析框架。所有阈值和权重可根据需要调整。

## 二、Dashboard 模块详解

### 模块 A — 顶部摘要卡片

显示当前核心指标的**最新真实值**：

| 卡片 | 数据来源 | 解读 |
|------|----------|------|
| 金价 | yfinance GC=F 最新收盘 | COMEX 黄金期货价格（美元/盎司） |
| 实际利率 | FRED DFII10 最新值 | **负值 = 强利好黄金**，正值且上行 = 压制黄金 |
| 美元指数 | FRED DTWEXBGS 最新值 | 美元强 → 压制金价，美元弱 → 利好金价 |
| 通胀预期 | FRED T10YIE 最新值 | Breakeven 越高，通胀对冲需求越强 |
| 核心 PCE 同比 | 计算值 | 美联储最关注的通胀指标，>3% 支撑黄金 |
| VIX | FRED VIXCLS 最新值 | >30 = 市场恐慌，避险资金流入黄金 |
| 联邦基金利率 | FRED DFF 最新值 | 加息周期 = 承压，降息/暂停 = 利好 |
| 美联储总资产 | FRED WALCL 最新值 | 扩表(QE) = 利好，缩表(QT) = 利空 |

**解读方式**：卡片上附带涨跌箭头和颜色，反映相对前值的变化方向。绿色/上涨不一定代表利好黄金——需对照具体指标（如实际利率上涨是利空黄金）。

### 模块 B — 历史走势图表

提供多个系列的近 1 年日度走势：

- **金价 + 均线** — 金价叠加 MA50/MA200 均线（虚线），观察趋势和交叉
- **实际利率 vs 金价** — 与金价反向关系是否成立？（左轴反转）
- **美元指数 vs 金价** — 观察美元与黄金是否呈负相关
- **通胀预期** — Breakeven 通胀率走势
- **美联储总资产 vs 金价** — QE/QT 流动性周期
- **VIX** — 恐慌指数是否触发避险
- **金银比** — 金价/银价比率，>80 为极端偏高，<60 为偏低
- **GDX vs 金价** — 金矿 ETF 与金价的相对表现
- **收益率曲线 (10Y-2Y)** — 负值=倒挂（衰退信号），趋势陡化/平坦化

**数据性质**：全部为 FRED/yfinance 真实历史数据。可切换 1M/3M/6M/1Y 时间范围。

**如何解读**：
- 金价上涨 + 实际利率下行 = 经典驱动组合，趋势可持续
- 金价上涨但实际利率也上行 = 可能是避险驱动（关注 VIX），不一定持久
- 美元与金价同向运动 = 异常，说明有其他强因子（如地缘政治）在主导

### 模块 C — 情绪仪表盘 & 雷达图

#### 情绪评分（0-100）

衡量市场**对黄金的乐观/悲观程度**（类似 Crypto Fear & Greed Index 思路）：

| 区间 | 标签 | 含义 |
|------|------|------|
| 0-20 | 极度恐惧 | 市场对黄金极度悲观，**可能是买入机会** |
| 20-40 | 恐惧 | 偏悲观 |
| 40-60 | 中性 | 多空平衡 |
| 60-80 | 贪婪 | 偏乐观，留意黄金过热风险 |
| 80-100 | 极度贪婪 | 市场对黄金极度乐观，**可能需要警惕回调** |

**⚠️ 自定义部分 — 6个因子权重**：

| 因子 | 权重 | 映射逻辑 | 理由 |
|------|------|----------|------|
| VIX | 15% | VIX 10→100分, 40→0分 | 低 VIX = 市场平静 = 黄金缺乏避险支撑 = 偏贪婪 |
| 实际利率趋势 | 25% | MA20−MA60 差值：−0.3→100, +0.3→0 | 利率下行 = 利好黄金 = 市场乐观 = 偏贪婪 |
| 美元趋势 | 20% | 20日 ROC：−3%→100, +3%→0 | 美元走弱 = 利好黄金 = 偏贪婪 |
| 技术面 | 10% | 四档位：15/40/65/90 | 多头排列 = 市场追涨 = 偏贪婪 |
| COT 持仓 | 15% | 百分位直接映射（0→0, 100→100） | 净多头百分位越高 = 多头越拥挤 = 偏贪婪 |
| GLD 资金流 | 15% | 5日流量：−20%→0, +20%→100 | 资金大幅流入 = 市场热情高 = 偏贪婪 |

> **注意**：情绪评分是**反向指标** — 极度贪婪时应警惕回调，极度恐惧时可能是买入窗口。

#### 雷达图（六轴，0-10）

每个轴的含义是**"对黄金的利好程度"**，分数越高 = 越利好：

| 轴 | 满分(10)条件 | 零分(0)条件 | 数据来源 |
|------|-------------|-------------|----------|
| 实际利率 | 实际利率 ≤ −1.5% | 实际利率 ≥ 2.5% | FRED DFII10 |
| 美元强弱 | 美元20日跌3%+ | 美元20日涨3%+ | FRED DTWEXBGS |
| 通胀预期 | Breakeven ≥ 3.5% | Breakeven ≤ 1.5% | FRED T10YIE |
| 流动性 | 美联储周扩表 ≥ +0.2% | 美联储周缩表 ≥ −0.2% | FRED WALCL |
| 避险需求 | VIX ≥ 40 | VIX ≤ 10 | FRED VIXCLS |
| 资金流入 | COT高百分位 + GLD大幅流入 | COT低百分位 + GLD大幅流出 | CFTC + yfinance |

**如何解读**：
- 雷达图形状越大、越饱满 = 多因子共振利好黄金
- 某个轴特别低 = 该因子当前不支持金价，是潜在风险来源
- 资金流入轴 = COT百分位(60%权重) + GLD资金流(40%权重) 合成

### 模块 D — 多空信号

信号引擎扫描当前所有因子，输出看多/看空信号列表。

#### ⚠️ 自定义部分 — 18条规则的阈值和强度

**看多信号**：

| # | 规则 | 触发条件 | 强度 | 逻辑 |
|---|------|----------|------|------|
| 1 | 实际利率为负 | TIPS < 0% | 5 | 黄金不生息，负实际利率下持有黄金零成本 |
| 2 | 实际利率下行趋势 | MA20 < MA60 | 4 | 利率下行周期利好黄金 |
| 4 | 美元走弱 | 20日 ROC < −2% | 3 | 金价以美元计价，美元弱则金强 |
| 6 | 核心 PCE > 3% | PCE YoY > 3% | 3 | 高通胀支撑黄金保值需求 |
| 8 | 美联储扩表 | 周环比 > 0 | 3 | QE 释放流动性利好黄金 |
| 10 | VIX > 30 | VIX > 30 | 3 | 恐慌情绪推高避险需求 |
| 11 | COT 投机净多头低位 | 百分位 < 20% | 3 | 空头拥挤，反弹概率高 |
| 12 | GLD ETF 资金大幅流入 | 5日流量 > +10% | 2 | 机构加仓信号 |
| 15 | 金价站上 MA200 | Price > MA200 | 3 | 中长期趋势偏多 |
| 17 | 50/200日均线金叉 | MA50 > MA200 | 5 | 经典多头信号 |
| 19 | 实际利率-金价背离（超卖） | Z-Score < −1.5 | 3 | 金价相对实际利率偏低，反弹机会 |
| 20 | 金银比偏低 | 金银比 < 60 | 2 | 均值回归特性，可能有上行空间 |
| 21 | 收益率曲线倒挂 | 10Y-2Y < 0 | 3 | 衰退风险支撑避险需求 |

**看空信号**：

| # | 规则 | 触发条件 | 强度 | 逻辑 |
|---|------|----------|------|------|
| 3 | 实际利率高位且上行 | TIPS > 2% 且趋势上行 | 4 | 高实际利率大幅压制黄金吸引力 |
| 5 | 美元走强 | 20日 ROC > +2% | 3 | 强美元压制金价 |
| 7 | 核心 PCE < 2% | PCE YoY < 2% | 2 | 低通胀减弱黄金保值需求 |
| 9 | 美联储缩表加速 | 周环比 < −0.05% | 3 | QT 收紧流动性 |
| 11 | COT 投机净多头拥挤 | 百分位 > 80% | 3 | 多头过于拥挤，回调风险 |
| 13 | GLD ETF 资金大幅流出 | 5日流量 < −10% | 2 | 机构减仓信号 |
| 16 | 金价跌破 MA200 | Price < MA200 | 4 | 中长期趋势转空 |
| 18 | 50/200日均线死叉 | MA50 < MA200 | 5 | 经典空头信号 |
| 19 | 实际利率-金价背离（超买） | Z-Score > 1.5 | 3 | 金价相对实际利率偏高，回调风险 |
| 20 | 金银比极端偏高 | 金银比 > 80 | 2 | 均值回归风险，银偏好可能回升 |
| 21 | 收益率曲线急陡化 | 10Y-2Y ROC > 0.3 | 2 | Risk-on 情绪，削弱避险需求 |

**综合信号判定**（自定义）：
```
bullScore = Σ(看多信号强度)
bearScore = Σ(看空信号强度)
差值 > 5  → 看多
差值 < -5 → 看空
否则      → 中性
```

**如何解读**：
- 关注**高强度信号**（4-5分）：金叉/死叉、实际利率极端值
- 多个中等信号(3分)共振 > 单个强信号
- 信号是**当前快照**，不预测未来。方向可能随数据更新翻转

### 模块 E — 风险矩阵

评估四类黄金投资方式的当前风险水平（低/中/高）。

| 投资方式 | 核心驱动因子 | 适合人群 |
|----------|-------------|----------|
| 实物黄金 | 长期通胀 + 央行购金 | 长期持有、保值需求 |
| 黄金 ETF | 实际利率 + 资金流 + 美元 | 中期配置、流动性好 |
| 黄金期货 | 利率预期 + COT持仓 + 波动率 | 短期交易、杠杆偏好 |
| 金矿股 | 金价趋势 + 股市环境(VIX) + 油价 + GDX表现 | 愿承受高波动、追求杠杆收益 |

**⚠️ 自定义部分**：风险打分用 if/else 加减分方式，基准分 50（矿股 55），± 5~20 调整。`<35=低风险, <65=中风险, ≥65=高风险`。

**如何解读**：
- 风险矩阵不是"推荐"，而是告诉你**每种工具当前的风险暴露**
- 期货和矿股天然波动更大，"中等风险"对它们来说已算温和
- 结合自身风险偏好使用：保守投资者在"高风险"时应减仓

### 模块 F — 走势预期

分三个时间维度给出方向判断：

| 维度 | 核心驱动 | 置信度依据 |
|------|----------|------------|
| 短期（1-4周） | 技术面：金叉/死叉、RSI、价格 vs MA200 | 信号一致性 |
| 中期（1-6月） | 宏观面：实际利率趋势、美元动量、收益率曲线 | 因子方向是否共振 |
| 长期（6月+） | 结构性因子：通胀趋势、5Y5Y远期通胀、央行购金 | 趋势持续性 |

**⚠️ 自定义部分**：
- 每条因子 +1 或 +2 分，bull > bear 则看多
- 置信度：差值 ≥3 = 高，≥1 = 中，0 = 低
- 文案为固定字符串（9 条预设摘要）
- **"央行持续购金"硬编码为长期看多 +1**（无实时数据，基于近年结构性趋势假设）

**如何解读**：
- 三个维度方向一致 = 强趋势信号
- 短期看空但中长期看多 = 可能是回调买入机会
- 置信度"低"= 信号矛盾，方向不明确

| 指标 | 来源 | 含义 |
|------|------|------|
| MA50/MA200 | 计算值 | 中长期趋势骨架 |
| RSI(14) | 计算值 | <30 超卖（可能反弹），>70 超买（可能回调） |
| 金叉/死叉 | MA50 vs MA200 | 金叉=中期趋势转多，死叉=转空 |

- RSI + MA交叉 配合看更可靠

## 三、新增数据：COT 持仓 & GLD ETF

### COT（Commitments of Traders）

**数据来源**：CFTC 官网 `https://www.cftc.gov/dea/newcot/deafut.txt`（当期）+ 年度历史 ZIP

**关键字段**：

| 字段 | 说明 | 本项目用法 |
|------|------|----------|
| Open Interest | 未平仓合约总数 | 分母，用于计算净多头占比 |
| NonCommercial Long | 投机多头 | 与 Short 相减 = 净多头 |
| NonCommercial Short | 投机空头 | |
| specNetLong | 投机净多头 = Long − Short | 核心指标 |
| specNetLongPercentile | 净多头在近3年的百分位排名 | 衡量仓位极端性 |

**如何解读**：
- **百分位 > 80%**：多头拥挤，历史上容易出现回调 → 看空信号
- **百分位 < 20%**：空头极端，物极必反 → 看多信号
- **20%-80%**：正常区间，不触发信号
- COT 每周二收盘统计，周五发布，有 3 天延迟

### GLD ETF

**数据来源**：yfinance `Ticker('GLD')`

| 字段 | 说明 | 本项目用法 |
|------|------|----------|
| totalAssets | GLD 总资产（AUM），美元 | 反映机构对黄金的配置规模 |
| flow5dPct | 近5日 vs 前5日的成交额(dollar volume)变化率 | 近似资金流入/流出 |
| navPrice | 每份 NAV | 参考 |
| volume | 当日成交量 | 参考 |

**如何解读**：
- `flow5dPct > +10%`：短期资金大幅流入，机构在加仓 → 看多信号
- `flow5dPct < −10%`：短期资金大幅流出，机构在减仓 → 看空信号
- **注意**：成交额变化不等于 ETF 份额变化（创建/赎回），是**近似指标**

## 四、数据可靠性等级

| 等级 | 类型 | 内容 | 可信度 |
|------|------|------|--------|
| ★★★ | 原始数据 | FRED 12个序列、yfinance 金价/油价/白银/GDX、CFTC COT、GLD AUM | **完全真实**，来自官方机构 |
| ★★☆ | 半手动数据 | WGC 央行购金季度数据 | **真实但需手动更新**，无免费 API |
| ★★☆ | 标准计算 | MA/RSI/ROC/波动率/同比/百分位 等 | **公式标准**，可复现 |
| ★☆☆ | 自定义模型 | 信号阈值、情绪权重、雷达映射、风险矩阵、走势预期 | **主观框架**，阈值可调 |

### 当前已知的硬编码假设

| 项目 | 位置 | 值 | 说明 |
|------|------|------|------|
| 央行购金趋势 | `compute_outlook()` | `cb_trend` 判断 | 基于 WGC 季度数据判断 increasing/stable/decreasing，已替代原硬编码 |
| 矿股基准风险偏高 | `compute_risk_matrix()` | `miner_score = 55` | 矿股天然杠杆属性，初始偏高风险 |
| 情绪评分阈值 | `compute_sentiment()` | ±0.3, ±3%, ±20% 等 | 线性映射区间，可根据历史回测调优 |
| 综合信号判定 | `compute_overall_signal()` | `diff > 5` / `< -5` | 阈值影响"看多/看空/中性"切换灵敏度 |

## 五、使用方法

### 环境准备

```bash
# 安装依赖
pip3 install requests numpy yfinance

# 设置 FRED API Key（免费申请：https://fred.stlouisfed.org/docs/api/api_key.html）
export FRED_API_KEY="your_api_key_here"
```

### 运行

```bash
cd /path/to/gold

# 拉取数据 & 生成 JSON
python3 fetch_data.py

# 启动本地服务查看 Dashboard
python3 -m http.server 8765
# 浏览器打开 http://localhost:8765
```

### 输出文件

- `data/dashboard.json` — 完整数据，前端读取此文件渲染所有模块

## 七、如何调整自定义参数

如果你对某些阈值/权重有不同判断，可在 `fetch_data.py` 中修改：

| 想调整的内容 | 修改位置 | 参数 |
|-------------|----------|------|
| 信号触发阈值 | `compute_signals()` | 各 `if` 条件中的数字 |
| 信号强度 | `compute_signals()` | `"strength": N` |
| 综合信号灵敏度 | `compute_overall_signal()` | `diff > 5` 改小更灵敏 |
| 情绪因子权重 | `compute_sentiment()` | `score_parts.append((score, 权重))` |
| 情绪映射区间 | `compute_sentiment()` | 各因子的 `max/min` 线性映射 |
| 雷达轴映射范围 | `compute_radar()` | 每个轴的 `(上界 − 值) / 区间 × 10` |
| 风险矩阵加减分 | `compute_risk_matrix()` | 各 `if` 分支的 `±N` |
| 走势预期权重 | `compute_outlook()` | 各因子的 `+1` / `+2` |

## 九、局限性与改进方向

1. **GLD 资金流为近似值**：用 dollar volume 变化率代替真实的 ETF 份额创建/赎回数据
2. **COT 延迟**：数据截止周二，周五发布，有 3 天滞后
3. **情绪模型未回测**：权重和映射区间基于经验设定，未进行历史回测验证
4. **无地缘政治因子**：避险需求仅通过 VIX 间接反映，缺少直接的地缘风险指标
5. **单一货币视角**：所有数据以美元计价，未考虑人民币金价等本币视角
6. **央行购金数据为硬编码季度值**：WGC 无免费 API，当前使用手动维护的季度数据，需定期更新
